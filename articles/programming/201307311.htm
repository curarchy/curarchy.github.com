IE8下table样式兼容问题

代码
<!DOCTYPE html PUBLIC "-//w3c//dtd html 4.0 transitional//en">
<html>
<head>
<meta content='IE=7' http-equiv='X-UA-Compatible'>
<link href="/webresources/bootstrap/css/bootstrap.min.css" rel="stylesheet" type="text/css" media="screen" />
<body>
	<style>
		.smp table{padding: 0px;width:100%;}
		.smp table td{vertical-align: bottom;border:1px solid #ccc;padding:10px;}
		.smp input{width:100%;margin:0;padding:0;}
	</style>

	<div style="width:300px;margin:20px auto;" class="smp">
			<table>
				<colgroup>
				<col style="width:33%;"/>
				<col style="width:34%;"/>
				<col style="width:33%;"/>
			</colgroup>
			<tr>
				<td>
					<input type="text"/>
				</td>
				<td>不正常</td>
				<td>bbb</td>
			</tr>
			<tr>
				<td>
					<input type="text" class="fixinput"/>
				</td>
				<td>正常</td>
				<td>bbb</td>
			</tr>
		</table>
	</div>
	<style>.fixinput{word-break:break-all;}</style>
</body>
</html>
出bug必要条件
<meta content='IE=7' http-equiv='X-UA-Compatible'>
<colgroup>
	<col style="width:33%;"/>
	<col style="width:34%;"/>
	<col style="width:33%;"/>
</colgroup>
input{*overflow:visible;}      // bootstrap.css 自带！
.smp table{width:100%;}
.smp input{width:100%;}
修复关键代码
<style>.fixinput{word-break:break-all;}</style>
