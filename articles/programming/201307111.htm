案例

用于快速选择航站，可以匹配三字码、拼音、英文、汉字筛选

ie6兼容 大数据量兼容 
<input id="filterInput" type="text" style="width: 100px;" />
初始化脚本
_$.auto.init($("#filterInput"), airport);
option参数
名称 类型 默认值 描述
popID string "autoDiv" 弹出层默认ID
optChar array ['@', '|', '^'] 数据源的分隔符、匹配符、非匹配符（尽量不要改）
pagination object 
{
pageSize: 10,
paginationSize: 5
}
 pageSize:每页个数，paginationSize：显示分页个数,请选择奇数
sort int 4 排序方式，0：原始排序，1：按中文，2：按英文，3：按拼音，4：按三字码
sortEnable bool true 是否允许排序。ie6、ie7建议关闭。插入以下代码即可。 
<!--[if lt IE 7]>
<script type="text/javascript">_$.auto.setup({"sortEnable":false});</script>
<![endif]-->
callback funcion null 回调函数
ie6下修复hoverbug
<!--[if lt IE 7]>
<style>body { behavior: url("Webresources/javascript/framework/csshover3.htc"); }</style>
<![endif]-->
数据源格式
var airport = "@-|中文名|英文名|拼音|三字码@-|中文名|英文名|拼音|三字码";
//匹配符用|,非匹配符用^
