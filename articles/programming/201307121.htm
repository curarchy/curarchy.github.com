案例

用于输入标签,空格、回车、tab、分号、右方向键代表一个tag输入完成

ie6兼容 
<div class="tag-container">
  <input type="text" id="pureTag" />
</div>
初始化脚本
_$.tag.init($("#pureTag"));
option参数
名称 类型 默认值 描述
repeatable bool "false" 是否允许键入重复
外部调用接口 代码 描述
tagSetup 
_$.tag.tagSetup(option);
 设置option参数
init 
_$.tag.init(obj, option);
 初始化
destory 
_$.tag.destory(obj);
 销毁
getValue 
_$.tag.getValue(obj,format);
 取值
setValue 
_$.tag.setValue(obj, values, sp);
 设值
add 
_$.tag.add(obj, text);
 添加值
remove 
_$.tag.remove(obj, text);
 删除值
